尝试搭建博客

发表于 2016-11-17

用Hexo搭建博客。

Hexo安装

  1. npm install hexo-cli -g
  2. 进入要初始化的目录，hexo init，npm install
  3. npm install hexo-deployer-git –save
  4. 配置_config.yml deploy: type: git repo:”repo地址” branch: master
  5. hexo new “postName” #新建文章
  6. 编辑postName.md文件 #写文章
  7. 将CNAME和favicon.ico等文件放入source文件夹中
  8. hexo g(enerate) #生成静态页面至public目录
  9. hexo d(eploy) #将.deploy目录部署到GitHub
  10. 替换掉加载慢的jQuery脚本
